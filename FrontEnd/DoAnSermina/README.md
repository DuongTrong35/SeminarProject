# Vĩnh Khánh Food Street – Automatic Food Tour Guide

This repository contains a responsive React frontend built with Vite for the "Vĩnh Khánh Food Street – Automatic Food Tour Guide" project.

## Features

- React with Vite (dev server: `npm run dev`)
- Responsive mobile-first layout
- Modern travel/food discovery design
- Navbar, hero, featured food cards, embedded map, and footer
- Pure HTML/CSS styling using Flexbox & Grid

## Getting Started

Install and start the development server:

```bash
npm install
npm run dev
```

The site will be available at `http://localhost:5173` (or the port shown by Vite).

Edit components under `src/components` to customize content and style.
